## ✅ 서비스 소개

혼자 여행하는 사람들에게 선호하는 테마를 기반으로 여행지를 추천하는 앱 서비스입니다. 영화, 드라마, 예능 등에서 나온 촬영지를 바탕으로 여행지를 추천해 줍니다.
